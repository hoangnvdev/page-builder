# Page Builder - User Guide

Welcome to the Page Builder! This guide will help you create your own website in minutes.

## What You Can Do

- ✅ Choose from professional templates
- ✅ Customize colors, text, and images
- ✅ See changes instantly
- ✅ Download a ready-to-use website
- ✅ No coding required!

## Getting Started

### Step 1: Select a Template

When you first open the Page Builder, you'll see a gallery of templates:

- **Modern Portfolio** - Perfect for personal portfolios, designers, freelancers
- **Business Landing** - Great for businesses, startups, product launches

Click "Select Template" on the one you like.

### Step 2: Customize Your Page

Once you select a template, you'll see:

1. **Preview Area (center)** - Your page as it will look
2. **Property Panel (right)** - Controls to customize your page
3. **Toolbar (top)** - Actions like Export and Reset

#### How to Edit:

**Editing Page Settings:**

- The Property Panel shows "Page Settings" by default
- Change font family, background color, text color
- These apply to your entire page

**Editing Sections:**

- Click any section in the preview (Hero, About, Projects, etc.)
- The Property Panel will update with that section's settings
- Edit text, colors, buttons, and more
- Click the ✕ button to return to Page Settings

### Step 3: Customize Content

#### Text Fields

- Click in the text box and type
- Changes appear instantly in the preview

#### Colors

- Click the color square to open a color picker
- Or type a hex code (like #0066cc)
- See the change immediately

#### Dropdown Menus

- Click to see options
- Select your preferred choice

#### Lists (Projects/Features)

- Edit each item's title and description
- Changes update automatically

### Step 4: Export Your Website

When you're happy with your page:

1. Click "Export HTML" in the top toolbar
2. Your page will download as an `.html` file
3. Open the file in any web browser - it works!
4. Upload it to any web hosting service to make it live

## Tips & Tricks

### Preview Your Changes

- Every change updates instantly
- No need to save - just export when ready

### Try Different Colors

- Experiment with color combinations
- Use the color picker or enter hex codes
- Common colors:
  - Blue: #0066cc
  - Green: #10b981
  - Purple: #8b5cf6
  - Red: #ef4444

### Reset If Needed

- Made too many changes? Click "Reset to Default"
- This brings back the original template settings

### Start Over

- Click "← Back to Templates" to choose a different template
- Your current changes will be lost

## Common Questions

**Q: Do I need to know how to code?**
A: No! Everything is point-and-click.

**Q: Can I use my exported page anywhere?**
A: Yes! The exported HTML file works in any web browser and can be uploaded to any web host.

**Q: Will my exported page look the same as the preview?**
A: Yes, exactly the same!

**Q: Can I edit the exported file later?**
A: You can edit it in the Page Builder again, but changes to the downloaded HTML require code editing. Best to finish in the builder first.

**Q: How do I make my site live on the internet?**
A: Upload your exported HTML file to a web hosting service like:

- GitHub Pages (free)
- Netlify (free)
- Vercel (free)
- Any web hosting provider

## Need Help?

If something doesn't work as expected:

1. Try clicking "Reset to Default"
2. Return to template gallery and select again
3. Refresh the page builder
