# @page-builder/ui

Reusable UI component library for the Page Builder application.

## Components

- **Button** - Versatile button component with variants
- **Card** - Card container with Image, Content, Title, Description, Actions
- **ColorPicker** - Color selection input
- **Divider** - Horizontal/vertical dividers
- **EmptyState** - Empty state placeholder
- **Flex** - Flexbox layout container
- **Grid** - Grid layout container
- **Input** - Text input component
- **Panel** - Side panel component
- **Select** - Dropdown select
- **Textarea** - Multi-line text input
- **Toolbar** - Top toolbar with left/right sections

## Usage

```jsx
import { Button, Card, Input } from "@page-builder/ui";
```

## Development

```bash
# Build the package
pnpm build

# Watch mode
pnpm dev
```
